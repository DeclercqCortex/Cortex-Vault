# Cortex seven-day trial

> Copy this file to your **vault** as `trial-notes.md` and fill in each
> day's section at the end of the day. Be honest. Petty complaints count
> — small friction compounds. The point of the trial is to surface
> issues before piling Phase 2's opinionated features on top of a weak
> foundation.

**Rules**

- Use Cortex for *all* note-taking for seven consecutive days. No Obsidian,
  OneNote, Apple Notes, or paper notebook for research work.
- When something annoys you, **write it down** in this file rather than
  tolerating it.
- If the app breaks, fix only if the fix takes under 30 minutes. Otherwise
  note the bug and work around it.
- Email and physical notes for non-research stuff are fine.
